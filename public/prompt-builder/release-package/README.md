# Prompt Builder Release Package

This folder contains the public-facing launch assets for Prompt Builder.

## Folder Map

- `docs/`: design philosophy, LinkedIn copy, press kit, launch checklist.
- `docs/CLI_INSTALL.md`: GitHub one-command install notes.
- `PICS/`: renamed visual candidates, usage report, and source image experiments.
- `PICS/archive/`: old visual candidates kept for reference, not public launch use.
- `PICS/PICS_USAGE.md`: placement report for landing page, PR package, GitHub README, and LinkedIn.
- `web-shots/`: rendered screenshots of the live local pages.
- `social-cards/`: 1200 x 630 post images for LinkedIn, GitHub, and portfolio sharing.
- `demo-notes/`: guidance for using the visual assets in launch posts.

## Best Default Launch Set

Use these four assets first:

1. `social-cards/post-launch.png`
2. `web-shots/landing-homepage.png`
3. `web-shots/builder-direct-mode.png`
4. `PICS/showcase-creative-range-what-can-you-build.jpg`
5. `PICS/preset-library-100-plus-catalog-infographic.jpg`

They cover the full story: product promise, real interface, core direct-mode proof, creative range, and preset depth.
