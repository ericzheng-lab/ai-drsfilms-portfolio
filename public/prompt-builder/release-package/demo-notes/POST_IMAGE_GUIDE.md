## What To Avoid

- Do not post text-only launch updates.
- Do not lead with raw code screenshots.
- Do not use only generated output images, because people need to see this is a real tool.
- Do not make the PWA/no-backend angle the headline. Use it as trust support.
- Do not use `../PICS/archive/legacy-*` files publicly.
- Do not use `../PICS/metrics-by-the-numbers-preset-stats.jpg` until the claims are rewritten.
