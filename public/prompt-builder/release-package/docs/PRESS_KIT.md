# Prompt Builder Press Kit

## One-Line Description

Prompt Builder is an installable AI image prompt tool that turns film and visual-production language into model-ready prompts.

## Three-Line Description

Prompt Builder helps creators build AI image prompts without starting from a blank text box.  
Users select workflows, fields, and chips based on real production vocabulary: shot size, lens, lighting, mood, aspect ratio, style, and format.  
The tool outputs prompts for Midjourney, Flux, Nano Banana, and GPT Image, with optional LLM enhancement.

## Founder / Author Bio

Eric Zheng is a filmmaker building AI tools for creators. He created Prompt Builder to help people skip the prompt-writing headache and focus on making things.

## Product Positioning

Prompt Builder is not a generic prompt library. It is a domain-specific interface for AI image generation.

Its core insight is that expert creative language can become an input system. Instead of making users learn model syntax, the product lets users express intent through the vocabulary they already understand.

## Suggested Launch Blurb

Prompt Builder is a new AI image prompt builder designed around film and visual-production language. It lets creators select shot sizes, lenses, lighting setups, moods, aspect ratios, and references, then converts those choices into prompts formatted for Midjourney, Flux, Nano Banana, or GPT Image. The app is static, installable as a PWA, can be launched with `npx github:...` or a one-command GitHub installer, and works without an account or backend.
