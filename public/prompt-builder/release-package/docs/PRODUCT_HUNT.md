**Prompt Builder**

## Tagline（一行）

Pick visual language, get a model-ready prompt. No prompt engineering required.

## 产品描述（Markdown）

Most AI image tools give you a blank box. Prompt Builder gives you a visual workflow.

**What it does:**
- Choose what you're making (mood board, storyboard, key art, character turnaround, panorama, UI mockup...)
- Select from 100+ curated presets: shot size, camera angle, lens, lighting, mood, style, aspect ratio
- Pick your model — Midjourney, Flux, Nano Banana, or GPT Image
- Copy a prompt already formatted for that model

**Why it exists:**
A cinematographer thinks in `epic wide shot`, `35mm anamorphic`, `blue-hour industrial glow`. Most prompt tools ask them to type into a blank box. Prompt Builder closes that gap.

**Key features:**
- 100+ curated film & visual production presets
- 4 model outputs: `/imagine` syntax, natural language, conversational, structured JSON
- Works directly in-browser — no account, no backend, no setup
- Installable as a desktop PWA
- Optional LLM enhancement when you want AI to expand your idea

**Tech:** Pure HTML/CSS/JS. Static. No dependencies. Runs from GitHub Pages.

Try it: [ericzheng-lab.github.io/prompt-builder](https://ericzheng-lab.github.io/prompt-builder/)

## Maker Comment（发布后第一条自评）

Hey PH 👋

I'm Eric, a filmmaker building AI tools for creators.

I built Prompt Builder because I got tired of watching talented visual people struggle with blank prompt boxes. A DP knows exactly what `anamorphic lens, teal-and-amber grade, man small in frame for scale` means — but most AI tools don't speak that language.

So I built a tool that does. Pick what you're making, select from real film vocabulary, and get a prompt formatted for whichever model you use.

It's a static PWA — no account, no backend, no data collection. Just open it and go.

Would love your feedback. What's missing? What would make you actually use this?
